# Disci-App 🗡️

Aplicación de gamification para crear y completar hábitos/misiones.

## Descripción
La Disci-App convierte tus hábitos diarios en misiones con sistema de XP y rangos. Crea misiones, asígnales dificultad y completa cada una para ganar puntos de experiencia.

## Rangos
| XP | Rango |
|---|---|
| 0–49 | 🌱 Novato |
| 50–99 | 🪄 Wizard |
| 100–199 | 🧙 Sorcerer |
| 200+ | 🔥 Archmage |

## XP por dificultad
- Fácil: 10 XP
- Normal: 25 XP
- Difícil: 50 XP

## Guía de instalación

1. Clona el repositorio:
```
git clone <repo-url>
cd disci-app
```

2. Abre `index.html` en tu navegador (doble clic) o sirve con un servidor local:

3. Con nginx, copia los archivos a `/var/www/html/disci-app/` y configura un bloque server.
